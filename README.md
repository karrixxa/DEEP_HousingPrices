# Housing Price Prediction

Constructing and pricing a home involves far more than square footage alone. Lot size, building type, year sold, neighborhood, amenities, and property features can all influence a home’s final sale price. This project explores how different housing characteristics relate to sale price and uses machine learning to predict home values from a wide range of property-level features.

Using a Kaggle housing dataset, this project investigates questions such as which building types tend to be most expensive, how home sale prices change over time, and how features like pools, lot area, and overall quality affect price. The workflow includes data cleaning, exploratory data analysis, feature engineering, model building, and performance evaluation.

As part of Rice University’s DEEP data science program, our team developed and presented this project in a Kaggle-style competition and earned **1st place**.

## Key Questions
- Which housing features are most strongly associated with sale price?
- Which building types tend to have the highest prices?
- How have home sale prices changed over time?
- Does having a pool meaningfully affect sale price?
- How accurately can machine learning models predict housing prices?

## Methods
- Data cleaning and preprocessing
- Exploratory data analysis

- Model evaluation and comparison
